## Instructions Include either `slidr.js` or `slidr.min.js` somewhere at the bottom of your html page, after the body content.
-------- ## HTML slidr works on any `inline`, `inline-block` or `block` elements with an `id` defined. Valid slides include any first-level children elements with the `data-slidr` attribute set to some unique value within the parent scope. For example:
  • apple
  • banana
  • coconut

banana
coconut
are all valid html markup for creating three different slidr's within the same page. (inline elements are transformed into inline-blocks in order to apply transitions).

------------ ## Javascript A global `slidr` object is available for calling. Creating a slidr can be as basic as this: slidr.create('slidr-img').start(); What happened? slidr finds the element with `id="slidr-img"`, and applies a `linear` transition effect in the `horizontal` direction to any direct children with the `data-slidr` attribute set. The first data-slidr child found is assumed as the starting slide, and by default the controls are shown along the `border` with a white `#fff` theme. Finally, the call to `start()` completes initialization and fades the element into the document.
