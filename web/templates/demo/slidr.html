-------- ## HTML slidr works on any `inline`, `inline-block` or `block` elements with an `id` defined. Valid slides include any first-level children elements with the `data-slidr` attribute set to some unique value within the parent scope. For example: ```html

  • banana
  • coconut

------------ ## Javascript A global `slidr` object is available for calling. The most minimal way of creating a slidr is this: ```javascript slidr.create('slidr-demo').start(); ``` The call to `create()` finds the element with `id="slidr-demo"`, applies the default settings, then returns a `Slidr` object. The call to `start()` converts all children elements with the `data-slidr` attribute set into slides. The first element found is assumed as the starting slide. Default settings will apply a `linear` transition effect in the `horizontal` direction, with a `fade` between slides and the overflow `hidden`. Slide controls are shown along the `border` using a white `#fff` theme. Finally, the Slidr element is faded into the document. `create()` accepts various settings in its second parameter. A call with all the settings toggled looks like so: ```javascript slidr.create('slidr-demo', { 'breadcrumbs': true, 'controls': 'corner', 'direction': 'vertical', 'fade': false, 'overflow': true, 'theme': '#222', 'transition': 'cube' }).start(); ```
------------ ## Settings Full details on available settings listed below: | Parameter | Type | Default | Description | | ------------- |:-------:|:------------:| -----------------------------------------------------------------------------:| | `breadcrumbs` | bool | `false` | Show or hide breadcrumbs on start(). `true` or `false`. | | `controls` | string | `border` | Show or hide control arrows on start(). `border`, `corner` or `none`. | | `direction` | string | `horizontal` | The default direction for new slides. `horizontal` or `h`, `vertical` or `v`. | | `fade` | bool | `true` | Whether slide transitions should fade in/out. `true` or `false`. | | `overflow` | bool | `false` | Whether to overflow transitions at slidr borders. `true` or `false`. | | `theme` | string | `#fff` | Sets color theme for breadcrumbs/controls. `#hexcode` or `rgba(value)`. | | `transition` | string | `linear` | The default transition to apply. `cube`, `linear`, `fade`, or `none`. |
